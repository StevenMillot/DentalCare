Politique de confidentialité

Cette page décrit comment les données personnelles sont traitées lorsque vous utilisez le site du Cabinet Parodontia.

1) Responsable du traitement

Cabinet Parodontia — Dr Raphaël Brochand (coordonnées dans les mentions légales).

2) Données collectées

  • Identité : prénom, nom
  • Contact : e-mail, téléphone
  • Demande : motif, message (optionnel)
  • Données techniques : logs serveur/plateforme d’hébergement (adresse IP, user-agent) à des fins de sécurité et de diagnostic

3) Finalités

  • Traiter vos demandes envoyées via le formulaire (contact / prise de rendez-vous)
  • Assurer la sécurité du site et prévenir les abus (anti-spam)

4) Base légale

  • Votre consentement lorsque vous envoyez volontairement le formulaire
  • Intérêt légitime : sécurité et prévention des abus

5) Destinataires / sous-traitants

Les informations saisies dans le formulaire sont transmises au Cabinet Parodontia. Selon la configuration, l’envoi du formulaire peut transiter par un prestataire technique :

  • FormSubmit (formsubmit.co) — service d’acheminement de formulaire vers e-mail
  • Web3Forms (web3forms.com) — optionnel si une clé d’accès est configurée

L’hébergement du site est assuré via GitHub Pages (voir mentions légales).

6) Durée de conservation

Les messages envoyés via le formulaire sont conservés le temps nécessaire au traitement de votre demande, puis archivés ou supprimés selon les obligations et besoins du cabinet. À défaut d’exigence particulière, une durée de quelques mois est généralement suffisante pour le suivi des demandes.

7) Vos droits

Conformément au RGPD, vous disposez notamment de droits d’accès, de rectification, d’effacement, de limitation et d’opposition, ainsi que du droit d’introduire une réclamation auprès de la CNIL. Pour exercer vos droits, contactez le cabinet via les coordonnées indiquées dans les mentions légales.

8) Cookies & contenus tiers

Le site n’utilise pas de cookies publicitaires ni d’outils de mesure d’audience de type “analytics” par défaut.

Une carte Google Maps peut être affichée sur la page d’accueil. Par défaut, la carte est désactivée : elle n’est chargée qu’après votre action (clic sur “Charger Google Maps”). Le chargement de Google Maps implique des échanges avec Google susceptibles d’entraîner des traceurs/cookies.

9) Contact

Pour toute question relative à la confidentialité, vous pouvez contacter le cabinet via contact@parodontia.fr.
